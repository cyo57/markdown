#### Linux 创始人

#### Linux常用发行版

- RHEL (Red Hat Enterprise Linux)
  由Red Hat公司提供的商业版本 Linux, 适用于需要商业保障的企业客户

- CentOS
  基于RHEL源码编译的社区发布版
  CentOS7维护期限到2024结束, 新版CentOS Stream为RHEL的上游开发分支, 滚动发行版

- Debian
  由开源社区维护, 注重稳定性和严格的包管理, 有丰富的软件源和社区

- Ubuntu
  基于Debian, 图形界面友好
