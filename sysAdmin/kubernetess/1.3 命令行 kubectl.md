## kubectl

### 资源操作

```shell
kubectl get no
kubectl get nodes

kubectl get po
kubectl get pod

kubectl get ns
kubectl get namespace
```

- 副本
```bash
kubectl get deploy
kubectl scale deploy --replicas=3 nginx
```

### Pod操作

- 创建 Pod
```shell
# --image：镜像
kubectl run nginx --image=nginx
```

- 创建 Service/Deploment
```shell
kubectl create <cmd>
# kubectl create --help
```
