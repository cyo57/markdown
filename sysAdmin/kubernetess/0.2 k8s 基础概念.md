## 特性

![[Container_Evolution.svg]]

- 服务发现和负载均衡
	k8s 可以使用 DNS 名称或者 IP地址 公开容器
	k8s 可以负载均衡并分配网络流量
- 存储编排
- 自动部署和回滚
- 自动装箱计算
	资源占用分配
- 自我修复
	k8s 动态迁移应用
- 密钥与配置管理
## 架构

### 工作方式

![](components-of-kubernetes.svg)
K8s Cluster = N Master Node + N Worker Node

高可用集群，Master 宕机时 Worker 投票决定新的 Master
